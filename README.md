# Project Overview
This project is a Pet Sitting and Pet Care Blog, deployed on an AWS EC2 Ubuntu instance using Apache as the web server.
The site provides helpful guidance for pet owners, including tips on nutrition, grooming, and trustworthy pet sitting services.
